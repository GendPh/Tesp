a) Implemente em TS uma função eVogal,
que receba uma string (só com um caracter) e retorne:
* true, se o caracter for uma vogal;
* false, caso contrário.

b) Implemente em TS uma função vogais,
que receba uma string e devolva uma
string só com as vogais da string recebida por parâmetro.
Use a função eVogal implementada anteriormente.

Exemplo de chamadas à função:
> vogais("ola")
> "oa"
> vogais("saltar")
> "aa"
> vogais("jogo")
> "oo"
> vogais("jjj")
> ""
