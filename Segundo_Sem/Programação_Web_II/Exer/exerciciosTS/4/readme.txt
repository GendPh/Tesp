Implemente em TS uma função capicua, que receba uma string e devolva um boleano:
* true, caso essa string seja uma capicua;
* false, caso contrário.

Uma capicua é uma palavra que lê da mesma forma para trás e para a frente.
Exemplo de chamadas à função:
> capicua('abba')
> true
> capicua('radar')
> true
> capicua('rir')
> true
> capicua('ola')
> false
